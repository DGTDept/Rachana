When you should use this component

You should use the pagination component when you want to display large amounts of related content that spans across multiple pages, such as search results or forum posts.

When you should consider something else

You should not use the pagination component to indicate progress, such as in a transaction.

You should also not use the component as a replacement for navigation. For main navigation, use the main nav and side nav components

Rounded Pagination buttons

You can use .is-rounded to make all pagination buttons rounded.
