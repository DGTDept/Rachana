Overview

Style

Modifier Description
sgds-modal is-active

Set the modal to display:block

(default value is display:none)
sgds-modal is-small

Set the modal width to 30vw (viewport width)
sgds-modal is-large

Set the modal width to 70vw (viewport width)
sgds-modal is-fullwidth

Set the modal width to 95vw (viewport width)

When to use

{{page.title}} should be used when:
  • You want to interrupt a user’s current task to direct the user’s attention to something important. e.g confirming when a user wants to submit a form. Or you may need user input such as a sign up or login without leaving the current page
{{page.title}} should not be used when:
  • You should not user the modal as a form of notification to the user. It should always accompany something actionable by the user such as confirming or filling in a form. If you need to notifiy the user or something important, do consider using the notification component instead.
