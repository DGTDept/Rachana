Handle different grid layouts for each screensize

Different column sizes per breakpoint

You can define a column size for each viewport size: mobile, tablet, desktop, widescreen and fullHD.

Class Breakpoint
is-x-mobile Mobile Up to 768px
is-x-tablet Tablet between 768px to 1023px
is-x-desktop Desktop between 1024px to 1215px
is-x-widescreen Widescreen between 1216px to 1407px
is-x-fullhd FullHD from 1408px and above
Resize the browser to see how the grid respond to different screensizes
