The container is an element designed to contain page elements to a reasonable maximum width based on the size of a user's screen. This is useful to couple with other UI elements like grids or menus to restrict content width to a reasonable size.

List of how the sgds-container will behave in different screensizes.

Screensize Container width
Desktop Maximum width of 960px
Widescreen Maximum width of 1152px
FullHD Maximum width of 1344px
Fluid Full width container that takes 100% of the screen size regardless of the screen width.
{{ default_container_example }}

If you do not want a maximum width but keep the 32px margin on both sides, add is-fluid.

{{ default_container_isFluid_example }}

The sgds-container class can be used in any context, but mostly as a direct child of either:

sgds-navbar sgds-hero sgds-section sgds-footer
