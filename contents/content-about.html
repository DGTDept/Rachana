What is Blueprint?

BLUEPRINT is a Design System which provides a framework used to simplify and ensure consistency across Singapore Government services. The system contains a collection of reusable UI Components which can be assembled together to build the applications of your choice and design. Guided by the Digital Services Standards, BLUEPRINT is crafted with the intention to help designers and developers to build government services easily.

What does BLUEPRINT solve?

Provides Better Scalability

Adaptable to suit the changing needs of your users, as you can build your own site with the UI Components provided.

Lower Overall Costs

A standard design system keeps your design and code overhead low.

Improves User Experience

Allows you to focus on developing a better user experience.

Faster Prototyping

Working within an exisiting design system allows you to put pieces together a lot faster when creating prototypes.

Easy Maintenance

Provides a common and easy to maintain architecture which eliminates inconsistent and non-reusable styles.

Provides Built-in Accessibility

Our UI Components are in compliance with the Digital Services Standards.

Our Design System

The Design Process

We studied the composition of interfaces and broke it down to fundamental building blocks. From the individual UI elements, we are able to combine and form a standard module which can assembled together to create a relatively complex section of an interface.


For designers, this would allow them to focus on the smaller building blocks rather than the whole site.


For developers, it simplifies the maintenance of styles and allows an effective on-boarding of new team members.
