Install with npm

npm is a package manager for Node based projects. Blueprint maintains a blueprint-govtech package for you to utilize the compiled files on your project
  1. Install Node.js here
  2. Run command $ npm install blueprint-govtech
  3. Link your stylesheet to /assets/css/main.css
