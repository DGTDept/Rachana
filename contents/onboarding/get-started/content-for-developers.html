There are different ways to use SGDS within your project. Which one you choose depends on the needs of your project and how you are most comfortable working.
Install with node package Install manually

Install with node package

npm is a package manager for Node based projects. SGDS maintains a package for you to utilize the compiled files on your project.
  1. Install Node.js here.
  2. To see if npm is installed, open your preferred command line tool and type in npm -v
    This should print the npm version number that look something like this 3.10.10.
  3. To install the sgds-govtech package, navigate to your site folder using your terminal
    and run this command npm install sgds-govtech --save.
  4. This will generate node_mouldes/sgds-govtech in your site folder.
    {%- highlight html -%} site/node_modules/sgds-govtech/ ├── _sass/ ├── css/ ├── fonts/ ├── js/ {%- endhighlight -%}
  5. The main SASS(scss) source file is located here site/node_modules/sgds-govetech/main.scss.
  6. Import the main.scss file into your own sass folder.
    Below is an example of how you might setup your main Sass file:
    {%- highlight sass -%} @import site/node_modules/sgds-govtech/main //your own css code.. {%- endhighlight -%}
  7. You can find more information here on how to compile SASS into CSS.

Install manually

We provide both minified CSS and the JavaScript files that you can use in a production environment or reduce the file size of your downloaded assets.
  1. Download SGDS package here.
  2. The zip package contains the sgds-govtech.css & sgds-govtech.jsfiles.
  3. Extract and copy the files and folders into a relevant place in your project’s code base.
  4. Reference the CSS and JavaScript files in each HTML page or templates in your project.
  5. For css, place the link tag between the html head tag. {%- highlight html -%} {%- endhighlight -%}
  6. For js, place the script tag between the html head tag. {%- highlight html -%} {%- endhighlight -%}
