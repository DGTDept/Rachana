For Designers

There are a few different ways to use Blueprint within your project. Which one you choose depends on the needs of your project and how you are most comfortable working.
UI Kit in Sketch
  1. Install Node/npm here
  2. Ensure the installation was done properly
  3. Create a package.json by running the command
  4. Add blueprint-govtech to your project's package.json
  5. You can now add the followings to your header
UI Kit in Invision
  1. Download Blueprint Package here
  2. Unzip the folder and put it in working directory
