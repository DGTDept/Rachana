Content Website Platform (CWP) is a common secured environment based on a resilient, robust and controlled platform which provides a suite of standardised software for hosting content-based websites. It enables unclassified government websites to be centrally managed and operated on public cloud, bringing the benefits of convenience, greater security, optimisation of resources, faster deployment speed and cost savings through economies of scale.

Hosting

CWP is hosted in the Public Cloud, Amazon Web Services (AWS) and G-Cloud, IM8 Compliant Hosting Environment, for the 'unclassified' front-end website and 'restricted' form services respectively.

Integrated, centrally-managed security

CWP centrally manages the security protection of all hosted websites, through a range of integrated security infrastructure and managements services.

At the perimeter, measures comprises:

  1. CyberWatch Centre, which captures and processes security alerts
  2. Content Delivery Network (CDN) for caching and distributing load
  3. Web application firewall to filter malicious web traffic
  4. Defacement monitoring, which monitors webpages against unauthorised changes

There is another layer of security which includes:

  1. Virtual Private Cloud to protect cloud resources
  2. End Point Protection against viruses and malware
  3. Network Intrusion Protection System to examine network traffic flows and block exploits
  4. Patch Management for monitoring and administering timely software patches, fixes and updates
  5. Cert Management for managing SSL/TSL certificates

Content publising services

CWP provides improved manageability and operational efficiency with five Content Management Systems (CMS) standardised software.

The five supported CMSes are:

  1. WordPress (with commercial support for plug-ins)
  2. SharePoint
  3. SiteCore
  4. Swiit
  5. Sitefinity
