Isomer Pages is perfect for small organisation sites. Think of it like a file-based CMS, without all the complexity. Our site generator takes your content, renders Markdown and Liquid templates, and spits out a complete static website ready to be served by any web server.
Hosting GitHub Pages

Hosting

Isomer Pages is using Jekyll, which happens to be the engine behind GitHub pages, so you can host your website on GitHub's servers for free.

Isomer Pages are meant for running static informational websites with 'unclassified' data. Websites should not be used for sensitive transactions like sending passwords or credit card numbers. It is subjected to the following usage limits:

  1. Source repositories have a recommended limit of 1GB
  2. Published websites may be no larger than 1GB
  3. Soft bandwidth limit of 100GB per month
  4. Soft limit of 10 builds per hour
