If you use Sass in your project, you can use your own custom Sass source in conjunction with SGDS to achieve a high level of customisation.

File Structure

You’ll have a file structure that looks like this if using a package manager like npm:

{%- highlight bash -%} site-folder/ ├── scss │ └── custom.scss └── node_modules/ └── sgds-govtech/sass/ └── sgds.scss {%- endhighlight -%}

If you’ve downloaded our source files and aren’t using a package manager, you’ll want to manually setup something similar to that structure.

{%- highlight bash -%} site-folder/ ├── scss │ └── custom.scss └── sgds-govtech/sass/ └── sgds.scss {%- endhighlight -%}
Importing SGDS.scss

{% highlight scss %} //Customisation starts here @import "site/youSassPath/customVariable"; //Customisation end here @import "site/node_modules/sgds-govtech/sgds/sass/sgds"; {% endhighlight%}
