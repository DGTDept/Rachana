Section is the main building block for all layouts. It can be used as direct children of body

section

Usage

You can change the overall padding sizes by adding these class names to your section:

  • is-small
  • is-medium
  • is-large
{% highlight html %}

Section

{% endhighlight %}

Style

Property Values
Default padding: 3rem 1.5rem
.is-small padding: 1.5rem 1.5rem
.is-medium padding: 9rem 1.5rem
.is-large padding: 18rem 1.5rem
