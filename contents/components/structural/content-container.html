Description

Container allows you to center your content horizontally on any screen size.

The .sgds-container class can be used in any context, but mostly as a direct child of either:

  • .navbar
  • .sgds-hero
  • .sgds-section
  • .sgds-footer
Default Container

If you do not want a maximum width but keep the 32px margin on both sides, add the is-fluid modifier.
