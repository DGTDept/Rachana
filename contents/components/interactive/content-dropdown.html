Guidelines

Dropdown menu should:

  • Use for secondary or less important information and actions since they are hidden until user exposes them by clicking on it
  • Contain actions that are related to each other

Style

Selector Description
sgds-dropdown the main container
sgds-dropdown-trigger the container for a button
sgds-dropdown-menu the toggable menu hidden by default
sgds-dropdown-content the dropdown box, with a white background and a shadow
sgds-dropdown-item each single item of the dropdown
sgds-dropdown-divider a horizontal line to separate dropdown items
