Get started with SGDS

Here you'll find an overview of the available components.

We highly recommend using the base SGDS classes and use a custom css file to override the styling if you need to. This is so that any styling updates to the base styles from us can be applied without you doing it manually. This also prevents any custom style you have create from being reverted back to the base styles.

For help with installation, do head on over to our installation guides section for the different ways you can install the design system.

For help with customisation, do head on over to our customise with sass section to see how you can customise the look and feel and behaviour of SGDS components.
