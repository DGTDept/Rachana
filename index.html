What is Blueprint?

Blueprint enables teams to build better products faster by making design reusable and makes scalable possible.

UI Components

UI Components are designed to unify for simplicity and consistency across government services with simple codes.

Layout Templates

Layout Templates are built with UI Components. Each template is flexible and can customise them for own needs.
