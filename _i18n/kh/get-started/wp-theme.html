1. Downloading the Theme

សន្មត់ថាអ្នកមានគេហទំព័រ WordPress បង្កើតឡើង។ ដើម្បីទាញយកកម្មវិធី Rachana Theme របស់យើងគ្រាន់តែចុចលើប៊ូតុងខាងក្រោម។ បន្ទាប់មក អ្នកនឹងទទួលបានឯកសារ rachana-theme.zip ។
Download
  • Version: 1.0.0
  • File Size: 501.9 kB
  • Compatible with: WordPress 5.8+

2. Logging into WordPress

  • ចូលប្រើ WordPress admin dashboard របស់អ្នក។
  • ចុចចូលទៅកាន់ > Themes

3. Installing the Theme

  • ចុចលើ "Add New Theme" នៅក្រោម Themes
  • Upload the rachana-theme.zip file
  • ចុច "Install Now"

4. Activating the Theme

  • បន្ទាប់ពីដំឡើងរួចរាល់ សូមចូលទៅកាន់ Appearance > Themes
  • ស្វែងរកប្រធានបទ Rachana Theme​​​ នៅក្នុងបញ្ជី
  • ចុច "Activate" ដើម្បីប្តូរទៅ Rachana Theme

5. Final looks

នៅពេលដែលអ្នកបានបើកដំណើរការ theme រួចរាល់ហើយ អ្នកអាចកំណត់ទិដ្ឋភាពផ្សេងៗនៃរូបរាងគេហទំព័ររបស់អ្នក។
