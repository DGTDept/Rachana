1. Downloading the Plugin

សន្មត់ថាអ្នកមានគេហទំព័រ WordPress បង្កើតឡើង។ ដើម្បីទាញយកកម្មវិធី Rachana Plugin របស់យើងគ្រាន់តែចុចប៊ូតុងខាងក្រោម។ បន្ទាប់មក អ្នកនឹងទទួលបានឯកសារ rachana-block.zip ។
Download
  • Version: 1.0.0
  • File Size: 88.5 kB
  • Compatible with: WordPress 5.8+

2. Logging into WordPress

  • ចូលប្រើ WordPress admin dashboard របស់អ្នក។
  • ចូលទៅកាន់​ Plugins

3. Installing the Plugin

  • ចុចលើ "Add New Plugin" នៅក្រោម Plugins
  • Upload the rachana-block.zip file
  • ចុច "Install Now"

4. Activating the Plugin

  • បន្ទាប់ពីដំឡើងរួច សូមចូលទៅកាន់ Plugins > Installed Plugins
  • ស្វែងរក Rachana Block នៅក្នុងបញ្ជី
  • ចុចលើ "Activate" ដើម្បីប្រើប្រាស់ Rachana Block

5. Final looks

នៅពេលដែលអ្នកបានបើកដំណើរការ plugin ហើយ ឥឡូវនេះអ្នកអាចប្រើ blocks របស់យើងដើម្បីប្ដូរគេហទំព័ររបស់អ្នកតាមបំណង។ នេះជាអ្វីដែលអ្នក គួរតែមើល៖
