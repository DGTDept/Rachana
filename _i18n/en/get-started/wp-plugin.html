1. Downloading the Plugin

Assumed you have a WordPress site set up. To download our Rachana Plugin, simply click the button below. Then, you will get the rachana-block.zip file.
Download
  • Version: 1.0.0
  • File Size: 88.5 kB
  • Compatible with: WordPress 5.8+

2. Logging into WordPress

  • Access your WordPress admin dashboard
  • Navigate to Plugins

3. Installing the Plugin

  • Click on "Add New Plugin" under the Plugins section
  • Upload the rachana-block.zip file
  • Click "Install Now"

4. Activating the Plugin

  • After installation, go to Plugins > Installed Plugins
  • Find Rachana Block in the list
  • Click "Activate" to utilize Rachana Block
