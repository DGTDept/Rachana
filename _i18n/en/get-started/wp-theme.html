1. Downloading the Theme

Assumed you have a WordPress site set up. To download our Rachana Theme, simply click the button below. Then, you will get the rachana-theme.zip file.
Download
  • Version: 1.0.0
  • File Size: 501.9 kB
  • Compatible with: WordPress 5.8+

2. Logging into WordPress

  • Access your WordPress admin dashboard
  • Navigate to Appearance > Themes

3. Installing the Theme

  • Click on "Add New Theme" under the Themes section
  • Upload the rachana-theme.zip file
  • Click "Install Now"

4. Activating the Theme

  • After installation, go to Appearance > Themes
  • Find Rachana Theme in the list
  • Click "Activate" to switch to the Rachana Theme

5. Final looks

Once you've activated the theme, you can customize various aspects of your website's appearance.
